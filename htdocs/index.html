Welcome Devy

I went ahead and got PostCSS running for you with cssnext included. I wanted to add more goodies, but I figured leaving that up to you would be more fun.

Suggested

  • cmder (if not using Linux)
  • XAMPP (if not using Linux)
  • Bootstrap
  • Node
  • Ruby and Ruby on Rails just for funzies
  • A helpful partner

Once you have Node going, you can just run `npm install` inside of the directory of this project. Once that is completed and all packages are installed, running `gulp watch` will compile the PostCSS.
